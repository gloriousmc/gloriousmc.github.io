Vanilla Server

Commands

This is a calm 100% command-block-run vanilla server. That means no confusing plugins or gimmicks. To us, vanilla is the flavor of being simple.

Although plugins were the easy way out, we wanted a challenge. That ment that we had to use Command Blocks. We had to use commands like:

/execute @a[team=hide1] ~ ~ ~ summon leash_knot ~ ~ ~ {CustomName:Center}
/scoreboard players set @a[score_hay=1,score_hay_min=1] hay 0


That's just a few of them. There are many more commands that make our server possible. In fact, there are even commands that you wouldn't even think are there.

Have you ever heard of /trigger? It's used to execute a scoreboard by a player.

/trigger grass set 1


/trigger was used in the making of the morph book for the hiders.

If you're thinking about trying out various commands and messing with command blocks, YouTube and Minecraft Forum are good places to start. To get a command block, just do /give @p command_block
